## What is HTML

```

<!DOCTYPE html> 
  <html>    
    <head>        
      <meta charset="utf-8">        
      <!--  CSS quote -->

      <title>Hello HTML</title>    
    </head>    
    <body>
      <h1> a line of text</h1>        
      <p>a paragraph</p>    
    </body>    
    <!--  JS quote --> 
  </html>
  ```
